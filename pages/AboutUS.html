About PlebDashboard

What is PlebDashboard?

PlebDashboard is an open source comprehensive data endeavor with focus on building data tools, products and dashboard for Lightning, Bitcoin and Bitcoin adjacent Technologies. Our mission is to provide insightful data and visualizations to help users understand and navigate the Bitcoin ecosystem.

Our Plan

  • Continue work on Lightning visualization. Expand our range of visualizations and metrics
  • Bring data and insights of Bitcoin and adjacent technologies
  • Build tools for data analysis. for example, LightningDS, a python library for data analysis for Lightning network
