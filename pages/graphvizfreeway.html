Node Visualization

Interactive Network graph visualization of all the nodes that are connected with Freeways. It plots only freeways - channels with capacity greater than 1BTC.

  • Edge width is proportional to channel capacity
  • Bubble Size is correlated with Node Capacity, and we have numebr of nodes attributes as tooltip
  • Select and move nodes to have a good view on the channels
  • Mulling over connection of Nodes will insight on how node may interact with each other.
